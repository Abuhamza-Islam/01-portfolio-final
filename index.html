Vanilla JavaScript

Vanilla JavaScript refers to plain JavaScript without any additional libraries or frameworks. It is used for building interactive and dynamic web applications directly in the browser.

React

React is a popular JavaScript library for building user interfaces, particularly single-page applications. It allows developers to create reusable UI components and manage state efficiently.

Node.js

Node.js is a runtime environment that allows you to run JavaScript on the server side. It is commonly used for building scalable and high-performance backend services and APIs.

MongoDB

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It is widely used for modern web applications due to its scalability and ease of use.
